## 2. Prisma de Kapferer

> Preencha cada faceta do prisma. As 6 dimensoes precisam ser coerentes entre si.
> Referencia: branding-masterclass.md, Secao 2.1

```
                        EMISSOR (a marca comunica)
         ┌──────────────────────────────────────────┐
         │                                          │
    PHYSIQUE (Fisico)              PERSONALITY (Personalidade)
    Elementos tangiveis            Tom, estilo, carater
    da marca                       da comunicacao
    ┌─────────────────┐           ┌─────────────────┐
    │                 │           │                 │
    │                 │           │                 │
    │                 │           │                 │
    └─────────────────┘           └─────────────────┘

    CULTURE (Cultura)              RELATIONSHIP (Relacao)
    Valores e crencas              Tipo de relacao com
    fundamentais                   o publico
    ┌─────────────────┐           ┌─────────────────┐
    │                 │           │                 │
    │                 │           │                 │
    │                 │           │                 │
    └─────────────────┘           └─────────────────┘

    REFLECTION (Reflexo)           SELF-IMAGE (Autoimagem)
    Como o publico se ve           Como o publico QUER
    ao usar a marca                se ver
    ┌─────────────────┐           ┌─────────────────┐
    │                 │           │                 │
    │                 │           │                 │
    │                 │           │                 │
    └─────────────────┘           └─────────────────┘
         │                                          │
         └──────────────────────────────────────────┘
                        RECEPTOR (o publico percebe)
```

**Perguntas-guia para cada faceta:**

| Faceta | Perguntas |
|---|---|
| **Physique** | Como a marca PARECE? Quais elementos visuais a representam? (cores, logo, tipografia, estilo fotografico) |
| **Personality** | Se a marca falasse, COMO falaria? Qual o tom? Que adjetivos descrevem sua personalidade? |
| **Culture** | Quais VALORES fundamentam a marca? Quais crencas sao inegociaveis? |
| **Relationship** | Que TIPO de relacao a marca tem com o publico? (mentor, amigo, parceiro, guia, professor) |
| **Reflection** | Como os USUARIOS se veem ao associar-se com a marca? ("Sou alguem que...") |
| **Self-Image** | Como os usuarios QUEREM se ver? ("Aspiro ser...") |

## 4. Positioning Statement

> Complete os templates abaixo. Use o que fizer mais sentido para seu contexto.
> Referencia: branding-masterclass.md, Secao 10

### Template 1: Formula Classica (Geoffrey Moore)

```
Para _______________________________________________
         [publico-alvo especifico]

que ________________________________________________
         [necessidade ou oportunidade]

__________________________________ e a/o _________________________
    [nome da marca]                     [categoria de mercado]

que ________________________________________________
         [beneficio unico / ponto de diferenca]

porque _____________________________________________
         [razao para acreditar / prova]
```

```
[Nome da marca] ajuda [publico] a [resultado desejado]
atraves de [metodo/abordagem unica].
```

### Template 3: Formula "Only" (Marty Neumeier)

```
[Nome da marca] e o unico [categoria] que [diferencial unico].
```

Preenchido:
_____________________________________________________

### Validacao do Posicionamento

- [ ] E especifico (nao generico)?
- [ ] E diferenciado (ninguem mais pode dizer o mesmo)?

- [ ] E crivel (posso comprovar)?
- [ ] E sustentavel (posso manter a longo prazo)?
- [ ] E memoravel (facil de lembrar)?

## 10. Brand Audit Checklist

> Use trimestralmente para verificar consistencia e saude da marca.

### Identidade Visual
- [ ] Logo sendo usado corretamente em todos os touchpoints
- [ ] Paleta de cores consistente (digital e impresso)
- [ ] Tipografia correta em todos os materiais
- [ ] Estilo fotografico alinhado
- [ ] Templates de conteudo atualizados

### Tom de Voz
- [ ] Tom consistente em todos os canais
- [ ] Vocabulario da marca respeitado
- [ ] Voice chart seguido por todos que criam conteudo
- [ ] Bio/descricoes atualizadas em todas as plataformas

### Conteudo
- [ ] Pilares de conteudo respeitados
- [ ] Frequencia de publicacao mantida
- [ ] Qualidade consistente (ou melhorando)
- [ ] Repurposing sendo feito sistematicamente
- [ ] CTA claro em todo conteudo

### Digital
- [ ] Site atualizado e funcional
- [ ] SEO basico implementado
- [ ] Links de bio atualizados
- [ ] Destaques do Instagram organizados
- [ ] YouTube otimizado (banner, descricoes, playlists)

### Metricas
- [ ] KPIs de branding acompanhados mensalmente
- [ ] Branded search monitorado
- [ ] Engajamento rate saudavel
- [ ] Lista de email crescendo
- [ ] NPS medido (se aplicavel)

### Alinhamento
- [ ] Posicionamento ainda e relevante?
- [ ] Publico-alvo ainda e o mesmo?
- [ ] Concorrencia mudou?
- [ ] Precisa de ajuste estrategico?
