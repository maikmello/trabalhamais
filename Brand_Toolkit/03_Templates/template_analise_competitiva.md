# Template de Analise Competitiva

> Use este template para analisar cada concorrente ou referencia de marca identificado na Fase 1 (Discovery).
> Crie um arquivo por player analisado (ex: `competitive_analysis_nomedamarca.md`).
> Ao final, consolide os insights no Brand Studio (secao 1.1).

---

## Instrucoes de Preenchimento

1. **Identifique 5-10 players** no nicho do cliente (concorrentes diretos + referencias de branding)
2. **Preencha um arquivo por player** usando a estrutura abaixo
3. **Classifique cada player** como "Concorrente Direto" ou "Referencia de Branding"
4. **Ao final**, crie a tabela comparativa e consolide no Brand Studio
